05 Materiais

O que precisamos de vocês

Para iniciar a produção, precisamos dos seguintes arquivos:

  • ✓
    Projeto arquitetônico completo
    DWG, RVT ou SKP — plantas, cortes e fachadas
  • ✓
    Plantas baixas das tipologias
    Com cotas e especificações de ambiente
  • ✓
    Implantação geral
    Paisagismo, acessos e áreas comuns
  • ✓
    Manual de marca / ID Visual
    Logo, paleta, tipografia do empreendimento
  • ✓
    Referências de acabamentos
    Pisos, revestimentos, metais, marcenaria
  • ✓
    Especificação de paisagismo
    Espécies, volumetria e layout externo
  • ✓
    Memorial descritivo
    Especificações técnicas gerais
  • ✓
    Fotos do terreno e entorno
    Contextualização do render
